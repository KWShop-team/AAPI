# How to inject:

```csharp
using AToolsAPI;

AAPI.setconfig("execname", "version", "execmessage");

AAPI.Inject(false); // True means console

```

# How to execute:

```csharp
using AToolsAPI;

AAPI.setconfig("execname", "version", "execmessage");

API.ExecuteScript(richTextBox1.Text); // Make sure to have richTextBox1 In UI!
```
